
### 本字體為簡體字自動轉繁體 
- 電子書 電子閱讀器 請下載 zen888-Regular-book-TW 或 zen888-SemiBold-book-TW使用<br>
- 電子書 電子閱讀器 偽直排需求 請下載 zen888-Regular-book-TW-90 或 zen888-SemiBold-book-TW-90  使用<br>
- 瀏覽器或創作用 請下載 zen888-Regular-write-TW 使用<br>
